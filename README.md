# Site De Mattia Advogados — Conceitos de Redesign

Apresentação dos **três conceitos de wireframe** para o novo site da
**De Mattia Advogados Associados** (Joinville/SC), desenvolvidos pela
**Agência Kamino**.

> Wireframes lo-fi em escala de cinza. As cores de marca, fotos e conteúdo
> final são placeholders — o objetivo nesta fase é validar **estrutura,
> hierarquia e jornada de conversão**.

## 🔗 Ver online

A apresentação está publicada via GitHub Pages:

**https://agenciakamino.github.io/site-demattia/**

A página inicial apresenta os três conceitos. Clique em cada card para abrir o
wireframe completo.

## 📐 Os três conceitos

| # | Conceito | Direção |
|---|----------|---------|
| 01 | **A Autoridade Serena** — `layout1-premium.html` | Minimalista premium, texto-protagonista, muito respiro |
| 02 | **Estratégico Executivo** — `layout2-moderno.html` | Hero dividido, tipografia display, assimetria e impacto |
| 03 | **Monografia Editorial** — `layout3-editorial.html` | Estética de revista, masthead, capítulos numerados |

## 🛠️ Stack

HTML estático + [Tailwind CSS](https://tailwindcss.com/) via CDN. Sem build —
basta abrir os arquivos `.html` no navegador.

## 📂 Estrutura

```
.
├── index.html              # Capa — seletor dos três conceitos
├── layout1-premium.html    # Conceito 01
├── layout2-moderno.html    # Conceito 02
└── layout3-editorial.html  # Conceito 03
```
